# このプロジェクトについて

**Fandom State Book** は、Markdown ベースの共同編集型ブックプロジェクトです。

- 原稿は GitHub の `contents/` 配下で管理されます。
- 変更は Pull Request でレビューされ、`main` への merge を契機に自動ビルドされます。
- ビルド済みの静的サイトは GitHub Pages で配信されます。

翻訳や修正提案は Issue / Pull Request で歓迎します。
